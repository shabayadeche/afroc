# AFROC Portal

This repository contains the current static AFROC website build based on the June 2026 implementation guideline.

## Included pages

- `index.html` - one-page public website
- `mdt-programme.html` - MDT archive and session library

## Design direction

The interface uses the AFROC guideline palette:

- Deep Green `#0B5E3C`
- Forest Green `#137A50`
- AFROC Yellow `#FFC107`
- Soft Yellow `#FFF4C7`

## Run locally

Open `index.html` in a browser, or serve the folder with any static file server.

## Current scope

- One-page public website for AFROC
- Searchable MDT archive interaction
- Programme, partner, event, and contact content

Convert this into a real application with:
