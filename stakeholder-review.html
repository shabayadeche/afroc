MDT Archive Snapshot

The archive now reads as a reference library instead of a loose summary

The MDT page has been reframed around real sessions, dates, specialty filters, named speakers, and direct recording references where available.

Archive strengths

  • Session-by-session archive with dates and attendance where available
  • Named presenters, moderators, and panelists
  • Direct recording reference links for multiple sessions
  • Search and filter tools by theme, specialty, and session period
Poster-supported archive narrative The archive is supported by real VTB materials and session content rather than generic summary text.
